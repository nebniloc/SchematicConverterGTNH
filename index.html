.schem to .schematic

GTNH Fork — Extended Block ID Support

This tool converts WorldEdit 1.13+ .schem files to legacy .schematic files for use with GregTech: New Horizons and its extended ID system.

You will be prompted to enter the block ID and metadata for any unmapped blocks. Toggle advanced tooltips F3+H in your GTNH instance to see block IDs and metadata.

IDs may shift when different add-ons are installed, so double-check that imported mappings are still correct.
